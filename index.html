CSS Visual Rules

CSS Fonts

Tag Name Description
<font-family> Font family Specifies the font of a text.
<font-size> Font-size sets the size of the text.
<font-weight> Font weight Specifies the weight of a font.

CSS Text

Tag Name Description
<text-align> Text align Align text to the element that holds it.
<color> Text colour Changes the colour of the text to a specified color name, HEX value or RGB value.
<line-height> Line height Used to specify the space between lines.
